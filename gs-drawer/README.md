# @carlos-gs99/gs-drawer

Drawer/Offcanvas side panel component with animations and accessibility.

## Installation
```bash
npm install @carlos-gs99/gs-drawer
```

## Usage
```tsx
import { GSDrawer } from '@carlos-gs99/gs-drawer';

<GSDrawer open={isOpen} onClose={() => setIsOpen(false)} title="Menu" placement="end">
  <p>Drawer content</p>
</GSDrawer>
```

## Features
- ✅ 4 placements (start, end, top, bottom)
- ✅ 4 sizes (sm, md, lg, xl)
- ✅ Animations
- ✅ Focus trap
- ✅ Keyboard navigation (ESC)
- ✅ Accessibility
- ✅ i18n (EN/PT)
