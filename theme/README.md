# @globalsoft/theme

Design tokens and theming system for Globalsoft Components.

## Installation

```bash
npm install @globalsoft/theme
```

## Usage

```tsx
import '@globalsoft/theme/tokens.css';
```

## Tokens Available

- Colors (Primary, Secondary, etc.)
- Spacing (xs, sm, md, lg, xl)
- Typography (font sizes, families)
- Border Radius
- Shadows
