# GSTooltip

Tooltip component with positioning and triggers.

## Installation

```bash
npm install @carlos-gs99/gs-tooltip
```

## Usage

```tsx
import { GSTooltip } from '@carlos-gs99/gs-tooltip';

<GSTooltip content="Tooltip text">
  <button>Hover me</button>
</GSTooltip>
```
