# GSCard

Card container component with variants.

## Installation

```bash
npm install @carlos-gs99/gs-card
```

## Usage

```tsx
import { GSCard } from '@carlos-gs99/gs-card';

<GSCard>
  <GSCard.Header>Title</GSCard.Header>
  <GSCard.Body>Content</GSCard.Body>
  <GSCard.Footer>Actions</GSCard.Footer>
</GSCard>
```
