# GSDivider

Divider component with orientation and text support.

## Installation

```bash
npm install @carlos-gs99/gs-divider
```

## Usage

```tsx
import { GSDivider } from '@carlos-gs99/gs-divider';

<GSDivider />
<GSDivider>Text</GSDivider>
<GSDivider orientation="vertical" />
```
